# 위플랩 룰렛 듀얼 타이머

위플랩 룰렛 후원 결과를 보고 시간을 빠르게 적용할 수 있는 OBS용 반자동 타이머입니다.

현재 버전은 **반자동 베타**입니다. 위플랩 후원알림 URL만으로 룰렛 결과가 자동 전달되는지는 아직 확인 단계라서, 기본 사용 방식은 스트리머나 매니저가 룰렛 결과를 보고 직접 입력하는 방식입니다.

## 주요 기능

- `멧돼지`, `돼지` 타이머 2개 독립 실행
- 타이머 이름 변경 가능
- `+30분`, `-1시간`, `돼지 +20분` 같은 입력 자동 계산
- 입력 성공 시 타이머 자동 시작
- OBS 브라우저 소스로 사용 가능
- 조작 패널 숨김 가능
- 위플랩 후원알림 URL 연결/디버그 확인 기능 포함

## 파일 구성

- `obs_timer.html`: 실제 타이머 파일
- `사용법.txt`: 간단 사용법

## OBS에 추가하는 방법

1. 이 저장소에서 `obs_timer.html` 파일을 다운로드합니다.
2. OBS에서 `소스 추가` → `브라우저`를 선택합니다.
3. `로컬 파일`을 체크하고 `obs_timer.html`을 선택합니다.
4. 화면 크기를 원하는 위치에 맞춥니다.
5. 조작 패널에서 타이머 이름과 시간을 설정합니다.

## 기본 사용법

룰렛 결과를 보고 입력창에 시간을 입력한 뒤 적용 버튼을 누릅니다.

예시:

```text
+30분
-1시간
돼지 +20분
멧돼지 +1시간
```

입력창에 이름 없이 시간만 적은 경우:

```text
+30분
```

이때는 `멧돼지 적용` 또는 `돼지 적용` 버튼을 눌러 적용할 대상을 선택합니다.

여러 결과를 한 번에 넣을 수도 있습니다.

```text
멧돼지 +1시간
돼지 -30분
```

이 경우 `전체 적용` 버튼을 누르면 각각의 타이머에 적용됩니다.

## 조작 패널 숨기기

OBS 화면에 조작 패널이 보이지 않게 하려면 URL 뒤에 아래 값을 붙입니다.

```text
?controls=0
```

예시:

```text
obs_timer.html?controls=0
```

패널을 다시 보고 싶으면 URL에서 `?controls=0`을 제거하거나, 화면에서 `H` 키를 누릅니다.

## 위플랩 URL 연결 테스트

위플랩 후원알림 URL을 입력하면 연결 상태를 확인할 수 있습니다.

현재 확인된 연결 메시지 예시:

```json
{"uniConnect":true}
```

이 메시지는 룰렛 결과가 아니라 위플랩 연결 확인 신호입니다.

실제 룰렛 후원 시 디버그 창에 다른 메시지가 뜨면, 그 내용을 기반으로 자동연동 기능을 추가할 수 있습니다.

## 현재 한계

- 위플랩 후원 테스트만으로는 룰렛 결과 이벤트가 발생하지 않을 수 있습니다.
- 실제 룰렛 후원 결과가 외부 메시지로 전달되는지는 아직 확인이 필요합니다.
- 따라서 현재 버전은 안정적인 **반자동 타이머**로 사용하는 것을 권장합니다.

## 배포 상태

임시 완성본 / 반자동 베타 버전입니다.
